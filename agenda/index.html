Elm for JavaScript Developers

Jack Franklin
Day 1, 30 June 2016  ·  15:20 - 15:50
Elm is a functional, statically typed language where all values are immutable.This provides a much more robust system that's also much easier to work with than the JavaScript equivalent. In this talk I'll introduce the Elm and demonstrate why I think Elm is something all JS developers should be paying attention to.

The Linguistic Relativity of Programming Languages

Jenna Zeigen
Day 2, 1 July 2016  ·  16:00 - 16:30
The hypothesis of linguistic relativity says that the languages we speak influence the way we think. I argue this applies to programming languages too. However, key differences between spoken languages and programming languages enable us to adapt and shift our ways of thinking more fluidly as we learn programming languages than as we learn spoken languages. We can and should leverage these differences to grow individually as programmers and further advance our favorite programming languages.
